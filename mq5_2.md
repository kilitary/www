### initialize

```python
mt5.initialize(path: str = "", server: str = "SERVER", login: int = LOGIN, password: str = "PASSWORD") -> bool
```

**Parameters**
| Name | Python Type | MQL5 Type | Description |
|------|-------------|-----------|-------------|
| `path` | `str` | `string` | Full path to the terminal executable (optional). |
| `server` | `str` | `string` | Server name (e.g., `"MetaQuotes-Demo"`). |
| `login` | `int` | `ulong` | Account login number. |
| `password` | `str` | `string` | Account password. |

**Description**: Establishes a connection to a running MetaTrader 5 terminal.

**Parameters**
| Name | Python Type | MQL5 Type | Description |
|------|-------------|-----------|-------------|
| `path` | `str` | `string` | Full path to the terminal executable (optional). |
| `server` | `str` | `string` | Server name (e.g., `"MetaQuotes-Demo"`). |
| `login` | `int` | `ulong` | Account login number. |
| `password` | `str` | `string` | Account password. |

**Return**: `bool` – `True` on success, `False` otherwise. Use `mt5.last_error()` for details.

**Example**
```python
import MetaTrader5 as mt5
if not mt5.initialize(login=12345678, password="myPass", server="MetaQuotes-Demo"):
    print("initialize() failed, error code =", mt5.last_error())
else:
    print("connected, terminal version", mt5.version())
```

## Notes

* All functions raise no Python exceptions; they return `None` or `False` on failure. Use `mt5.last_error()` for diagnostics.
* The structures (`AccountInfo`, `SymbolInfo`, `TradeRequest`, etc.) are simple objects with attributes matching the MQL5 fields.
* Time values are returned as **seconds since epoch** (`int`). Convert with `datetime.fromtimestamp()` when needed.
* Constants such as `mt5.TIMEFRAME_M1`, `mt5.ORDER_TYPE_BUY`, `mt5.COPY_TICKS_ALL` are defined in the module.
